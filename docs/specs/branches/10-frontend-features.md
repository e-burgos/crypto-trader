# Spec 10 — Frontend Features (Config, News, Analytics, Admin, WS, Notifications)

## Objetivo

Agregar panel de configuración, news feed, analíticas, admin, WebSocket y sistema de notificaciones.

## Alcance

- Config panel: thresholds, API keys, sandbox/live
- News feed: últimas noticias, sentimiento
- Analytics: win rate, drawdown, Sharpe
- Admin panel: kill-switch, user mgmt
- WebSocket: integración tiempo real
- Notificaciones: toasts, centro

## Criterios de aceptación

- Todas las features integradas y funcionales
- Branch: `feature/frontend-features`
